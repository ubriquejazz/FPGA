This directory contains the most recent anomaly lists for new products.
The naming convention is:
AAAA Anomalies [Vx.xxx] UMMDDYY

Where:
AAAA = part, e.g. HH6U
Vx.xxx = Build Revision (optional - may not be present)
UMMDDYY = date of last update to list (Update month, day, year).

E.g. HHPC Anomalies V1.431 U121800
    = list for Hammerhead PCI Build Revision 1.431 Last Update Dec 18, 20000
